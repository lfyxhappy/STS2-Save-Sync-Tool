为 STS2 Mod 玩家准备的小工具

工具只处理进度存档，不修改游戏本体、mod、联机协议或官方存档结构。

普通档与 Modded 档对比

扫描 `%AppData%\SlayTheSpire2\steam\<steamId>` 下的 profile1-3，并展示两侧进度摘要。

手动同步方向

工具不会自动判断哪边更新，也不会自动覆盖。你明确点击“同步到 Modded”或“同步到普通档”。

Steam 缓存同步

同步 progress.save 的同时更新 Steam 本地缓存镜像和 remotecache.vdf 元数据。

安全说明

建议关闭游戏后再同步，避免游戏运行中写回旧存档。同步前会自动备份目标侧旧文件。

备份位置

`%AppData%\SlayTheSpire2\save_sync_tool\backups`

支持文件

当前版本只处理 `progress.save`，并同步对应 backup、Steam cache 和 remotecache 元数据。

运行环境

Windows 桌面程序，当前发布包依赖 .NET 8 Desktop Runtime。

FAQ

Will it overwrite my save automatically?

No. You choose the sync direction manually, and the target side is backed up before writing.

Does it support multiple profiles?

Yes. It scans profile1, profile2, and profile3 for the selected Steam account.

Does it change game files or mods?

No. It only works with progress.save and related local cache metadata.

Why update Steam local cache?

Because Steam Cloud may use local cache metadata. Updating the cache mirror and remotecache.vdf helps reduce stale cache write-back issues.
